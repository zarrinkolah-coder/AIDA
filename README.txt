نسخه آماده بروزرسانی روزانه از سرور

تغییر اصلی:
- برنامه هنگام باز شدن به صورت خودکار فایل data/latest.json را از کنار خودش/سرور می‌خواند.
- کاربر لازم نیست هر روز فایل JSON را در موبایل انتخاب کند.
- برنامه آخرین داده موفق را در حافظه گوشی نگه می‌دارد تا اگر اینترنت یا سرور در دسترس نبود، همان اطلاعات قبلی نمایش داده شود.
- دکمه «بروزرسانی از سرور» برای دریافت فوری فایل جدید اضافه شده است.
- بارگذاری دستی فایل JSON همچنان برای مواقع اضطراری باقی مانده است.

روش استفاده روزانه:
1. یک‌بار این پوشه mobile-web را روی هاست، GitHub Pages یا سروری که برنامه از آن باز می‌شود قرار دهید.
2. از روز بعد فقط فایل JSON خروجی نرم‌افزار مدیریت تولید را با نام latest.json داخل پوشه data جایگزین کنید:
   data/latest.json
3. کاربران با باز کردن برنامه، اطلاعات جدید را خودکار دریافت می‌کنند.

نکته مهم:
- فایل data/latest.json عمداً داخل این ZIP قرار داده نشده تا فایل نمونه خالی، اطلاعات واقعی کاربران را پاک نکند.
- فایل نمونه برای الگو اینجاست: data/latest.example.json
- فایل service-worker طوری تغییر کرده که latest.json کش نشود و همیشه نسخه تازه خوانده شود.


نسخه manifest اصلاح‌شده برای PWA Builder:
- حذف favicon.ico از manifest
- اضافه شدن description، screenshots، orientation و id
- فایل روزانه همچنان data/latest.json است.
